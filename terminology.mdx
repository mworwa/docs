## Agent
An Agent is an AI-powered entity designed to analyze content based on specified criteria. Agents can be customized to address various needs, such as detecting toxicity, spam, or inappropriate content.

## Label
Every Agent is build using Labels. Label contains criteria which will be predicted by Agent.

## Query
A Query refers to a single request made to the API for content analysis. Each time you request an Agent to analyze a piece of content it counts as one Query.
