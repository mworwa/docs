## 4. Make API Request

<CodeGroup>

```bash cURL
curl --request POST \
  --url https://gardianai.com/api/v1/agents/{agentId}/analyze/text \
  --header 'Content-Type: application/json' \
  --header 'x-api-key: <api-key>' \
  --data '{
  "content": "I don'\''t like your products"
}'
```

```python Python
import requests

url = "https://gardianai.com/api/v1/agents/{agentId}/analyze/text"

payload = {"content": "I don't like your products"}
headers = {
    "x-api-key": "<api-key>",
    "Content-Type": "application/json"
}

response = requests.request("POST", url, json=payload, headers=headers)

print(response.text)
```

```javascript Javascript
const options = {
  method: 'POST',
  headers: {'x-api-key': '<api-key>', 'Content-Type': 'application/json'},
  body: '{"content":"I don\'t like your products"}'
};

fetch('https://gardianai.com/api/v1/agents/{agentId}/analyze/text}', options)
  .then(response => response.json())
  .then(response => console.log(response))
  .catch(err => console.error(err));
```
